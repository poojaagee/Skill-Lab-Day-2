# BUILD AND DEPLOY FLASK APPLICATION TO AZURE WEB APP

Tools Used:
-Visual Studio Code(IDE For Python)
-Microsoft Azure
-GitHub Repository
-Python(Flask)

PROJECT OUTLINE:
1. Create requirements.txt: Lists dependencies (like Flask) needed to run your app.
2. Write Flask app (.py file): Your actual app code goes here.
3. Create GitHub Actions workflow (.yml file): Automates build & deploy process.
4. Add Azure secrets to GitHub: Stores sensitive info (like Azure credentials) securely.
5. Update .yml with Azure login & deploy steps: Tells GitHub Actions how to deploy to Azure.
